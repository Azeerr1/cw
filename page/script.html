Un script shell est un programme informatique développé pour fonctionner dans un shell Unix, un interpréteur de commandes.
Sur la plupart des systèmes Linux, un programme appelé bash (qui signifie Bourne Again SHell, version améliorée du programme shell Unix d'origine, sh, écrit par Steve Bourne) agit en tant que programme shell. ... Tous les shell peuvent exécuter des commandes situées dans un fichier.
